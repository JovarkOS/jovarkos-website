Welcome to JovarkOS,

A GNU/Linux distribution based on Arch Linux with the goal of providing stability and speed for those who are comfortable using a computer. It is built by teens, for teens.

Get Started Download Installer

Built for your needs.

Smart Defaults

We include the most popular GNOME extensions out of the box - some of our developers own the popular extensions themselves!

Immersive Desktop Experience

Speed and agility are the names of the game here. Delve into your work with Vivaldi or browse securely using Brave, whichever you choose.

Latest software

At its core, Arch Linux fuels JovarkOS's software repository, plus we add some of our own via convenience scripts and custom repository.

Fast, for everyone!

We make sure to compile and install the latest stable low-latency kernel for the ultimate desktop experience, even on old hardware!
